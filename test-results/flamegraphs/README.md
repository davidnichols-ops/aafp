# Flamegraphs

This directory holds flamegraph SVGs generated by `scripts/flamegraph.sh`
(Track M2 — Flamegraph integration).

## Generating a flamegraph

From the `implementations/rust/` directory:

```bash
# Single benchmark
./scripts/flamegraph.sh mcp_transport_ping
./scripts/flamegraph.sh handshake
./scripts/flamegraph.sh framing
./scripts/flamegraph.sh discovery

# All benchmarks
./scripts/flamegraph.sh --all

# List available benchmarks
./scripts/flamegraph.sh --list
```

## Output files

Each run writes `<benchmark-name>.svg` into this directory. Open the SVG in a
browser for an interactive flamegraph (click to zoom, search by function name).

## Platform requirements

- **Linux:** `perf` (kernel perf_event support). You may need to relax
  `perf_event_paranoid`: `echo 1 | sudo tee /proc/sys/kernel/perf_event_paranoid`.
- **macOS:** DTrace via `cargo flamegraph`. Requires `sudo` and SIP adjusted
  (`csrutil enable --without dtrace` from macOS Recovery). See the script's
  header comment for full instructions.

## Prerequisites

Install the profiling tool:

```bash
cargo install flamegraph
```
